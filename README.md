## ğŸ”‘ API Usage

This project uses the **Jamendo Music API**, which provides access to a large catalog of Creative Commons licensed music.

### What the API is used for:
- Fetching artists
- Fetching tracks
- Searching by name
- Filtering by genre
- Streaming and download URLs

### Important Notes:
- The API **does not guarantee artist country data** for all artists
- Country filtering works **only when metadata is available**
- All downloads link directly to Jamendo-hosted files

ğŸ” API credentials are required and must be obtained from:
